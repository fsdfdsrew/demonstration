## 1. Сервис управления кофейными автоматами

**Тип:** e-commerce (B2C через терминалы), административная панель  
**Стек:** Vue3, Node.js, PostgreSQL, FastAPI  
**Роль:** разработка фронтенда, проектирование API, координация команды  

### Функционал:
- Выбор напитков, оплата, статус заказа (терминал)
- Админ-панель: управление ассортиментом, ценами, статистика продаж
- Интеграция с платёжной системой

## 2. Сервис для ЖК (заявки и обращения)

**Тип:** B2B/B2C, портал для жителей и управляющей компании  
**Стек:** Vue2, Fastify, PostgreSQL  
**Роль:** разработка личных кабинетов, проектирование БД, логика ролей  

### Функционал:
- Личный кабинет жителя: подача заявок, отслеживание статуса
- Кабинет администратора: обработка заявок, назначение исполнителей
- Уведомления о статусе
- Чат в реальном времени (WebSocket)
- Интеграция с телефонией

## 3. Backend часть интернет-магазина

**Тип:** e-commerce (веб-каталог)  
**Стек:** FastAPI, Redis, Celery, Docker, Nginx, 1С: Предприятие  
**Роль:** разработка бэкенда и координация

### Функционал:
- Каталог товаров с фильтрацией, пагинацией, поиском
- Корзина, оформление заказа
- Админка для управления товарами
- Интеграция с 1С
- Кеширование через Redis
- Подключение метрик и мониторинга
